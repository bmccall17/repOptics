# Knowledge handoff to academy

When meaningful work is completed (ship, major fix, failure/postmortem, key decision):
- write a short learning note into `academy (brain)`:
  - what happened
  - why it happened
  - what we’ll do next time
  - links to PR/commit/issues if available

Default location:
- `academy/lessons-learned/YYYY-MM-DD_<topic>.md`
